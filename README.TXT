Es gibt ein (unsichtbares) Interface 'GrafikObjekt', das man als Ersatz fuer eine geminsame 
'Super-Klasse' (Ober-Klasse) aller grafisch sichtbaren Objekte ansehen kann.
Man findet dieses Interface als Typ in Methoden, die ein anderes GrafikObjekt uebergeben bekommen.
z.B. beruehrt(...) , berechneAbstandX(...) , berechneAbstandY(...)
